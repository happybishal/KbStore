Ecommerce Store

A fully-featured products e-commerce store built with Django, Alpine.js, and TailwindCSS.

Users can:
- Browse products by category
- Search for products live
- Add/remove items from a cart (with persistent state)

---

Tech Stack
- Backend: Django  
- Frontend: Alpine.js  
- Styling: TailwindCSS  
- Deployment: Render
